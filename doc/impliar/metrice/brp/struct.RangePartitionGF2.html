Expand description

A struct that calculates an fs::FSelect instance that fits f32_vec with its corresponding binary_labels (0|1).

The search for this solution can be one of:

  • greedy approach (see RangePartitionGF2::greedy_search__decision)
  • brute-force approach (see RangePartitionGF2::brute_force_search__decision)

Fields

f32_vec: Array1<f32>

target vector of f32, elements in [0,1]

binary_labels: Array1<usize>

vector of equal size to <f32_vec>, each i’th element is 0 or 1 and corresponds to i’th element of <f32_vec>

size_threshold: usize

number of partitions that fselect solution can have

fselect: FSelect

solution

score: usize

score of solution

contra_indices: Vec<usize>

indices in f32_vec such that fselect solution contradicts label in binary_labels.

fs_cache: Vec<FSelect>

cache of candidates in brute-force|greedy approach

c_type: String

basic|fm

description

calculates an std. score for the fs::FSelect f by observing its performance on the subvector index range r of f32_vec. Formula is: (number of partitions of f) * (number of elements of f32_vec[r.0..r.1] that contradict its binary label)
